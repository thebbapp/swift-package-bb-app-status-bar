# ``BbAppStatusBar``

Status bar and toast notification management.

## Overview

The BbAppStatusBar library provides a status bar component for displaying temporary notifications and status messages to users. It includes the status bar view, message queuing, automatic dismissal timing, and customizable styling options. This component is used throughout the application to provide non-intrusive feedback about operations, errors, and state changes.
